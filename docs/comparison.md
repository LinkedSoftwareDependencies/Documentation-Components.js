| Configuration       | External JSON-LD | Annotations | Programmatic | External JSON | Programmatic | 
| Component discovery | ✓                |             |              |               |              |
| Semantic            | ✓                |             |              |               |              |
| Compilation         | ✓                | ✓           |              |               |              |
